Shopping_list.java is a straightforward little console app I put together. It uses a standard array with a 5-slot limit to keep things simple. I set up a while loop that handles the data entry—it basically keeps the conversation going until the list is full or the user decides they’re done. After that, it loops back through to print everything out cleanly. I also tossed in a quick linear search at the end, so you can actually check if an item made it onto the list without scrolling back up.
